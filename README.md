# Shu-Bittorrent-Client

## Overview

Shu-Bittorrent-Client is a lightweight and efficient BitTorrent client designed for simplicity and ease of use. It supports downloading and seeding torrents with minimal system resource consumption. 

## Features

- **Peer-to-Peer File Sharing:** Download and upload torrents with ease.
- **Magnet Link Support:** Easily add torrents using magnet links.
- **Torrent Queue Management:** Control the order of your torrents.
- **Bandwidth Throttling:** Limit the bandwidth usage to prevent network congestion.
- **Cross-Platform Support:** Compatible with Windows, macOS, and Linux.
- **Simple and Clean UI:** Intuitive interface for managing torrents.
